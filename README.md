# `midid` - basic midi synthesiser

## building
requires:
*  `tokei`
*  `pkg-config`
*  `gcc`
*  `libjack2`
*  `clang-tools`
*  `gnumake`
*  `linenoise`
*  `lldb`
*  `bear`
*  `gdb`

```sh
make all # build project 
make run # run project
```

use qpwgraph or jackctl to wire up MIDI devices to synth, and audio to audio output
